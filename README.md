# 🌐 Personal Portfolio Website (Standard Version)

## 🚀 Live Demo

👉 https://portfolio-standard-navy.vercel.app/

---

## 📌 About This Project

This is my **standard-level portfolio website** built to showcase my skills, projects, and experience as a web developer.
It includes enhanced UI/UX, dynamic sections, and improved performance compared to the basic version.

---

## ✨ Features

* 🎨 Modern and responsive design
* ⚡ Fast performance with Next.js
* 🌙 Dark / Light mode support
* 📂 Projects section (with demo & code links)
* 📝 Blogs section (dynamic articles showcase)
* 📜 Certificates & education sections
* 📩 Contact form with real email sending (Nodemailer + Gmail)
* 📱 Fully optimized for mobile devices

---

## 🛠️ Tech Stack

* **Frontend:** Next.js, React, Tailwind CSS
* **Backend:** Next.js API Routes
* **Email Service:** Nodemailer (Gmail SMTP)
* **Deployment:** Vercel

---

## 📧 Contact Feature

The contact form is fully functional and sends emails directly to my inbox using Nodemailer.
Users can send messages from the website without relying on third-party services like EmailJS.

---

## 🚀 Deployment

This project is deployed using **Vercel**.
Every push to GitHub automatically triggers a new deployment.
